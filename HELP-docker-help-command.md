# Docker Help Command
This document shows you how to use the docker help command on every docker command.

## General help Command

**Command:**
```bash
docker --help
```

**Output:**
> Shows the top-level Docker help with all available management commands and subcommands.


## Docker help on any command

**Syntax:**
- Command:
  ```bash
  docker <command> --help
  ```

  > **Output:** This shows the help / usage information for any specific Docker command.

**Example:**

- Help on docker run command
  ```bash
  docker run --help
  ```
  
  > **Output:** This prints the help output for the docker run command, showing options, flags, and usage examples.

- Help on docker ps command:

  ```bash
  docker ps --help
  ```
  
  > **Output:** This prints the help output for the docker ps command.
